---

####  Iniciando aplicação:

* Clone o repositório:
  * `git clone git@github.com:IgorHLeal/ignite-lab-design-system.git`
---

* Instale as dependências do projeto:
  * `npm install`
---

* Dê um 'start' na aplicação:
  * `npm run dev` -
---

* Dê um'start' no storybook:
  * `npm run storybook`
